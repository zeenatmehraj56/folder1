list of selectors in CSS

  • Universal selector

      • Its symbol is * .
      It is used when we want to keep any similar property to the whole document .
  • Elementary selector
      • It does not require any type of symbol .
      It is used when we want to keep different properties individually. like {h1,p,body etc}
  • Id selector
      • Here we make a unique id for a particular element .
      At the time of styling we just select that partcular id and start styling.
  • Last one is class selector
      • It is similar with id but we can't keep same id to multiple elements but in class selector it is possible..
